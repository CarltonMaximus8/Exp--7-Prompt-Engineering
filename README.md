Exp:7-  Reproducing an Image Using Prompts for Image Generation

Aim:

To demonstrate the ability of text-to-image generation tools to reproduce an existing image by crafting precise prompts.

Objective:

To identify key elements within a reference image and use prompt engineering techniques to generate an image as close as possible to the original image using AI image generation tools.

## Initial Prompt

> Draw a butterfly sitting on flowers in a garden. Include clouds and the sun. Create it as a black-and-white coloring page.

### Expected Initial Output

* Butterfly and flowers present
* Basic garden background
* Simple coloring-page appearance
* Limited similarity to the original image

---

## Refined Prompt

> Create a high-quality black-and-white coloring book page for children. A large smiling cartoon butterfly with decorative wings is positioned in the center above several large flowers. Include rolling hills, fluffy clouds, patches of grass, and a bright sun in the sky. Use thick, clean outlines with no shading, no colors, and plenty of open spaces for coloring. The illustration should be cheerful, simple, detailed, and suitable for kids. Portrait orientation, coloring-book style, vector line art.

### Expected Final Output

* Larger butterfly
* Detailed wings
* Proper garden scenery
* Clean coloring-book appearance
* Much closer to the original image

# Conclusion

The experiment demonstrated that prompt engineering greatly improves image generation quality. Initial prompts produced generic results, whereas refined prompts containing detailed information about objects, background, lighting, style, colors, and composition generated outputs much closer to the reference images. Therefore, carefully structured prompts are essential for achieving high-quality and accurate AI-generated images.

**Result:** Refined prompts generated images with significantly higher similarity to the reference images compared to basic prompts, proving the effectiveness of prompt engineering in AI image generation.
